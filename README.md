# Image Processing CLI

A command-line image processing tool developed as part of the Agile Development Processes course.

## Project Overview
This project provides a lightweight CLI for basic image manipulation tasks such as:
- Image format conversion
- Resolution adjustment
- Blurring, sharpening, and recolouring

The application focuses on automation, data safety, and usability for developers and content creators.
